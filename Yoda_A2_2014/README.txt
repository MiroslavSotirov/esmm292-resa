To prepare the each of the systems follow the instructions from the assignment:
3. Open a command line prompt and go to the directory where you unzipped the archive.
4. Compile all of the applications by typing: javac *.java
4_2. Compile the system specific sources by typing: javac Task{1,2,3_2}/*.java
5. Type rmic MessageManager to build the Message Manager interface stubs.
6. Before you start the Message Manager or Museum Environmental Control System,
make sure that port 1099 is available on your system.

To run the system first start the manager through the EMStart.bat and SC{1,2,3}Start.bat respectively.
